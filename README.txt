React LOTR api est un essai de consommation d'api en react.js
Voici le lien vers le site fournissant l'api : https://the-one-api.dev/
Pour faire fonctionner l'app correctement, il faut vous inscrire ici : https://the-one-api.dev/sign-up et utiliser la clé API fournie.
La clé api est à coller dans un fichier nommé "apikey.txt". Ce fichier est à mettre dans le dossier src de l'app.

Sans cette clé, le seul endpoint fonctionnant correctement est le suivant : "/book", seul le 1er icone génèrera le contenu associé.
Voir https://the-one-api.dev/documentation pour la liste complete des endpoints.
